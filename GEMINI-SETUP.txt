# Как подключить Google Gemini (бесплатно)

## Шаг 1 — Получить ключ

1. Откройте: https://aistudio.google.com/apikey
2. Войдите в Google-аккаунт
3. Нажмите **Create API key**
4. Скопируйте ключ (начинается с `AIza...`)

## Шаг 2 — Вставить в проект

Откройте файл `server/.env` и добавьте:

```
GEMINI_API_KEY=AIzaSy...ваш_ключ...
AI_PROVIDER=gemini
AI_MODEL=gemini-2.5-flash
```

## Шаг 3 — Перезапуск

Закройте окно сервера и снова запустите **START.bat**

## Проверка

- Откройте **AI-репетитор** в приложении
- Вверху должно быть: `gemini / gemini-2.5-flash`
- Напишите: «Объясни алфавит простыми словами»

## Лимиты (бесплатно)

- ~1500 запросов в день
- Без банковской карты
- Для одного ученика — более чем достаточно
